Q1 Probability axioms, sum & product rules, conditional probability, Bayes' theorem.

⚡ Core Intuition

Probability counts favourable outcomes ÷ total outcomes, capped between 0 and 1.

Sum rule = "OR" (add, subtract overlap); product rule = "AND" (multiply, condition if dependent).

📖 Concept Explained
  • Axioms: P(A) ≥ 0, P(S) = 1, P(A∪B) = P(A)+P(B) if disjoint.
  • Sum rule (general): P(A∪B) = P(A) + P(B) − P(A∩B).
  • Product rule: P(A∩B) = P(A)·P(B|A) = P(B)·P(A|B).
  • Independent: P(A∩B) = P(A)·P(B); conditional drops.
  • Bayes': flips conditional — go from P(E|H) to P(H|E).
🔗 Connects to Random variables Joint distribution Naive Bayes Total probability
📐 Exam Answer Structure
  • State 3 axioms (Kolmogorov).
  • Venn diagram for A∪B, A∩B.
  • Sum + product + Bayes formulas in a box.
  • One numerical (disease/test or balls/urn).
AXIOMS (Kolmogorov)
  1) P(A) ≥ 0
  2) P(S) = 1
  3) Disjoint: P(A∪B) = P(A) + P(B)

SUM RULE  : P(A∪B)  = P(A) + P(B) − P(A∩B)
PRODUCT   : P(A∩B)  = P(A) · P(B|A)
CONDITIONAL: P(A|B) = P(A∩B) / P(B),  P(B) > 0
BAYES     : P(H|E)  = P(E|H)·P(H) / P(E)
💡 Example

Card from 52-deck: P(Heart ∪ King) = 13/52 + 4/52 − 1/52 = 16/52 = 4/13.

BAYES NUMERICAL — Disease test
Given: P(D)=0.01, P(+|D)=0.95, P(+|¬D)=0.05
Find : P(D|+)

P(+) = P(+|D)P(D) + P(+|¬D)P(¬D)
     = 0.95·0.01 + 0.05·0.99
     = 0.0095 + 0.0495 = 0.059

P(D|+) = 0.0095 / 0.059 ≈ 0.161

Answer: Only ~16.1% — base rate matters.
Mnemonic: "OR adds, AND multiplies. Bayes flips the arrow."
✅ Model Exam Answer

  2. Axioms — non-negativity, normalisation, additivity.
  3. Diagram — Venn with A, B, A∩B labelled.
  4. Rules — sum, product, conditional, Bayes (boxed).
  5. Numerical — disease/test Bayes solved (0.161).
  6. Significance — foundation for every distribution and inference.

Q3 PMF, PDF and CDF — define, compare, and show graphically.

⚡ Core Intuition

PMF = probability mass at exact points (discrete). PDF = density curve (continuous; area gives prob).

CDF = cumulative — integral/sum of PMF/PDF up to x.

📖 Concept Explained
  • PMF: p(x) = P(X=x), Σ p(x) = 1, p(x) ∈ [0,1].
  • PDF: f(x) ≥ 0, ∫ f(x)dx = 1; f(x) is not probability — area is.
  • CDF: F(x) = P(X ≤ x); always 0 → 1, monotone non-decreasing.
  • Discrete CDF: step function; Continuous CDF: smooth curve.
  • P(a<X≤b) = F(b) − F(a) — works for both.
🔗 Connects to Distribution function Expectation E[X] Binomial Normal
📐 Exam Answer Structure
  • Define PMF, PDF, CDF (one line each).
  • Two diagrams: discrete (stems + step CDF), continuous (curve + smooth CDF).
  • State F(b) − F(a) interval rule.
PMF  : p(x) = P(X = x),  Σ p(x) = 1
PDF  : f(x) ≥ 0,         ∫ f(x) dx = 1 over support
CDF  : F(x) = P(X ≤ x)
        = Σ_{t≤x} p(t)     (discrete)
        = ∫_{-∞}^{x} f(t) dt (continuous)

INTERVAL: P(a < X ≤ b) = F(b) − F(a)
DERIV  : f(x) = dF(x)/dx (where differentiable)
💡 Example

Fair die: p(k)=1/6 for k=1..6; F(3)=3/6=0.5; F(6)=1.

Mnemonic: "Mass at point (PMF), Density over range (PDF), Cumulative up to x (CDF)."
✅ Model Exam Answer
  1. PMF — discrete; p(x)=P(X=x); Σ=1.
  2. PDF — continuous; area = probability; ∫=1.
  3. CDF — F(x)=P(X≤x); 0→1 monotone.
  4. Diagram — 4-panel: PMF/step CDF, PDF/smooth CDF.
  5. Rule — P(a<X≤b)=F(b)−F(a).

Q4 Properties of a distribution function F(x).

⚡ Core Intuition

The CDF is a "running total" of probability. It starts at 0, climbs only upward, and ends at 1.

📖 Concept Explained
  • P1 — Monotone non-decreasing: x₁ < x₂ ⇒ F(x₁) ≤ F(x₂).
  • P2 — limx→−∞ F(x) = 0.
  • P3 — limx→+∞ F(x) = 1.
  • P4 — Right-continuous: F(x⁺) = F(x).
  • P5 — 0 ≤ F(x) ≤ 1 always.
🔗 Connects to CDF PMF/PDF Quantile

  • Sketch a representative CDF curve (S-shape or step).
  • One-line justification per property.
PROPERTIES OF F(x)
  1. 0 ≤ F(x) ≤ 1
  2. F is non-decreasing:  x₁ < x₂ ⇒ F(x₁) ≤ F(x₂)
  3. lim_{x → -∞} F(x) = 0
  4. lim_{x → +∞} F(x) = 1
  5. F is right-continuous: lim_{h→0⁺} F(x+h) = F(x)
  6. P(a < X ≤ b) = F(b) − F(a)

Q5 Binomial distribution — PMF, mean, variance + worked example.

⚡ Core Intuition

Repeat a yes/no trial n times with fixed success probability p. Count successes. That count is Binomial(n, p).

📖 Concept Explained
  • Fixed n trials, each independent, two outcomes (S/F).
  • P(success) = p constant across trials.
  • PMF: P(X=k) = C(n,k) pk (1−p)n−k, k=0..n.
  • Mean = np; Variance = np(1−p); SD = √(npq).
  • Symmetric when p=0.5; skewed otherwise.
🔗 Connects to Bernoulli Poisson (n→∞, p→0) Normal (n large)
📐 Exam Answer Structure
  • 4 conditions (independence, fixed n, two outcomes, constant p).
  • PMF formula boxed.
  • Mean and variance derived/stated.
  • Numerical: coin tossed 10 times, P(X=7).
BINOMIAL  X ~ Bin(n, p)
  PMF : P(X = k) = C(n,k) · p^k · (1−p)^(n−k),  k = 0..n
        C(n,k)  = n! / [k!(n−k)!]
  Mean: E[X]   = n·p
  Var : Var(X) = n·p·(1−p)
  SD  : σ      = √(np(1−p))

CONDITIONS (BINS):
  B inary outcome  · I ndependent trials
  N fixed          · S ame p each trial
WORKED — Coin tossed 10 times, P(7 heads)?
n = 10, p = 0.5, k = 7

C(10,7) = 10!/(7!·3!) = 120
P(X=7)  = 120 · (0.5)^7 · (0.5)^3
        = 120 · (0.5)^10
        = 120 / 1024
        ≈ 0.1172

Answer: P(X=7) ≈ 0.1172 or 11.72%.
💡 Example

Quality control: 5% defective; sample 20. X = defectives → Bin(20, 0.05); E[X]=1.

Mnemonic: "BINS — Binary, Independent, N fixed, Same p."
✅ Model Exam Answer
  1. Definition — n trials, two outcomes, fixed p.
  2. PMF — C(n,k) p^k q^(n−k).
  3. Mean/Var — np and np(1−p).
  4. Diagram — bar chart, peak at np.
  5. Numerical — 10 tosses, P(7H) = 0.1172.
  6. Use — defects, surveys, dice counts.

Q6 Poisson distribution — derive PMF, mean, variance + worked example.

⚡ Core Intuition

When rare events arrive at a steady rate λ per unit (time/space), count them. That count is Poisson(λ).

📖 Concept Explained
  • Limit of Binomial as n→∞, p→0 with np=λ constant.
  • PMF: P(X=k) = e−λ·λk/k!, k=0,1,2,…
  • Mean = Variance = λ (key property; useful exam check).
  • Events independent, rate constant, no two events same instant.
  • Skewed right for small λ; ≈ Normal for large λ.

Q7 Geometric distribution — PMF, mean, variance + worked example.

How many trials until your first success? That count is Geometric(p) — counts the wait.

Q8 Uniform distribution — discrete & continuous; mean & variance.

⚡ Core Intuition

Every outcome equally likely — flat probability over a finite range. Maximum ignorance / no preference.

📖 Concept Explained
  • Discrete uniform: X ∈ {x₁,…,xₙ}, P(X=xᵢ) = 1/n.
  • Continuous uniform: X ∈ [a,b], f(x) = 1/(b−a).
  • Continuous mean = (a+b)/2; var = (b−a)²/12.
  • CDF (continuous): F(x) = (x−a)/(b−a) for a ≤ x ≤ b.
  • P(c < X < d) = (d−c)/(b−a) within [a,b].
🔗 Connects to Random sampling Monte Carlo CDF inverse method

Q10 Normal (Gaussian) distribution — PDF, properties, 68-95-99.7 rule + z-table problem.

⚡ Core Intuition

The bell curve. Sums of many small effects converge here (CLT). Symmetric around μ; spread set by σ.

📖 Concept Explained
  • Continuous; parameters μ (mean), σ (SD > 0).
  • PDF: f(x) = (1/(σ√2π)) e−(x−μ)²/2σ².
  • Symmetric, unimodal, asymptotic to x-axis.
  • Standardise: Z = (X−μ)/σ ~ N(0,1).
  • Empirical rule: ≈68% within μ±σ, 95% within μ±2σ, 99.7% within μ±3σ.
🔗 Connects to CLT Binomial & Poisson approx Z-table Hypothesis testing
📐 Exam Answer Structure
  • PDF formula + parameters.
  • Bell curve diagram with σ-bands shaded.
  • 68-95-99.7 box.
  • Z-standardisation formula.
  • Numerical: marks ~ N(μ,σ²), P(X > 70) via z-table.
NORMAL  X ~ N(μ, σ²)
  PDF : f(x) = (1 / (σ√(2π))) · e^[−(x − μ)² / (2σ²)]
  Mean: μ
  Var : σ²

PROPERTIES
  · Symmetric about μ ;  mean = median = mode
  · Bell-shaped, asymptotic to x-axis
  · Area = 1 (total probability)
  · Inflection points at μ ± σ

STANDARDISATION
  Z = (X − μ) / σ  ~  N(0, 1)
  P(a < X < b) = Φ((b−μ)/σ) − Φ((a−μ)/σ)

  P(μ − σ  < X < μ + σ ) ≈ 0.6827

🚨 Common mistakes (red flags)

  • Confusing discrete Poisson with continuous Exponential — they're related but distinct (Poisson counts events; Exponential measures wait time between them).
  • Writing P(X=c) > 0 for a continuous RV — for continuous RVs, P(X=c) is always 0; probability lives in intervals.
  • Forgetting the −P(A∩B) correction in the general sum rule (only drop it when A,B are disjoint).
  • Using p for "failure" in Geometric — convention is p = success probability and X starts at 1, not 0.
  • Forgetting to scale λ when Poisson window changes (3/hr ≠ 3 for 2 hr — use λ'=6).
  • Mixing μ and σ² when writing N(μ, σ²) — note the parameter is σ², not σ.
  • In Bayes, computing P(E) without total probability theorem — must sum over all hypotheses.
  • Treating PDF value as a probability — it's a density; only the integral (area) is a probability.

Q1 Expectation E[X] — definition and properties

⚡ Core Intuition

Expectation is the long-run average — weight each outcome by its probability and sum.

📖 Concept Explained
  • Discrete: E[X] = Σ x·P(X=x).
  • Continuous: E[X] = ∫ x·f(x) dx.
  • Linearity: E[aX + bY] = a·E[X] + b·E[Y] — always, even if X,Y dependent.
  • Constant: E[c] = c.
  • Product rule: E[XY] = E[X]·E[Y] only if X ⊥ Y.
  • Var(X) = E[X²] − (E[X])².
🔗 Connects to moments variance LLN MGF
📐 Exam Answer Structure
  • Define expectation for discrete + continuous.
  • State 4 properties (linearity, constant, product, non-negativity).
  • Prove linearity by sum/integral.
  • Solve a worked numerical.
💡 Example

Fair die: E[X] = (1+2+3+4+5+6)/6 = 3.5 — the average roll over many trials.

Given: pmf  P(X=1)=0.1, P(X=2)=0.2, P(X=3)=0.4, P(X=4)=0.2, P(X=5)=0.1
Find:  E[X] and Var(X)

Step 1: E[X]  = 1(0.1)+2(0.2)+3(0.4)+4(0.2)+5(0.1)
             = 0.1+0.4+1.2+0.8+0.5 = 3.0
Step 2: E[X²] = 1(0.1)+4(0.2)+9(0.4)+16(0.2)+25(0.1)
             = 0.1+0.8+3.6+3.2+2.5 = 10.2
Step 3: Var(X) = E[X²] − (E[X])² = 10.2 − 9 = 1.2
Answer: E[X] = 3.0, Var(X) = 1.2

Q2 Moments — raw, central, standardized

⚡ Core Intuition

Moments are summary numbers — each higher moment captures a finer shape feature: mean, spread, skew, peakedness.

📖 Concept Explained
  • Raw moments (about origin): μ'ₙ = E[Xⁿ].
  • Central moments (about mean): μₙ = E[(X−μ)ⁿ].
  • Standardized moments: μₙ / σⁿ — dimensionless.
  • μ'₁ = μ = mean (location).
  • μ₂ = σ² = variance (spread).
  • μ₃ / σ³ = skewness (asymmetry).
  • μ₄ / σ⁴ = kurtosis (tail-heaviness, peakedness).
  • Relations: μ₂ = μ'₂ − μ'₁², μ₃ = μ'₃ − 3μ'₁μ'₂ + 2μ'₁³.

Q3 Moment Generating Function Mₓ(t) — definition + Binomial proof

⚡ Core Intuition

MGF is a "fingerprint" of a distribution — if two distributions share an MGF, they are identical, and differentiating MGF at t=0 yields the moments.

📖 Concept Explained
  • Definition: Mₓ(t) = E[e^(tX)], wherever the expectation exists.
  • Discrete: Mₓ(t) = Σ e^(tx) P(X=x).
  • Continuous: Mₓ(t) = ∫ e^(tx) f(x) dx.
  • Series form: Mₓ(t) = 1 + μ'₁ t + μ'₂ t²/2! + μ'₃ t³/3! + …
  • n-th moment: μ'ₙ = M⁽ⁿ⁾ₓ(0) — n-th derivative at t=0.
  • Hence E[X] = M'ₓ(0) and Var(X) = M''ₓ(0) − [M'ₓ(0)]².
🔗 Connects to moments Binomial CLT characteristic fn
📐 Exam Answer Structure
  • Define MGF (both forms).
  • State 3 properties (uniqueness, linearity, sum).
  • Derive Mₓ(t) for Binomial(n,p).
  • Differentiate at 0 to get mean and variance.
💡 Example

Binomial(n,p): Mₓ(t) = (q + p·eᵗ)ⁿ — direct from binomial theorem.

DERIVE MGF + MEAN + VARIANCE OF BINOMIAL(n,p)

X ~ Bin(n,p),  P(X=k) = C(n,k) pᵏ qⁿ⁻ᵏ,  k=0..n
Mₓ(t) = E[e^(tX)]
      = Σ_{k=0}^n  e^(tk) · C(n,k) pᵏ qⁿ⁻ᵏ
      = Σ_{k=0}^n  C(n,k) (p·eᵗ)ᵏ qⁿ⁻ᵏ
      = (q + p·eᵗ)ⁿ           ← Binomial theorem

Step 1 — Mean E[X] = M'ₓ(0)
M'ₓ(t)  = n (q + p·eᵗ)ⁿ⁻¹ · p·eᵗ
M'ₓ(0)  = n (q + p)ⁿ⁻¹ · p · 1   = n · 1 · p = np      ✓

Step 2 — Second moment E[X²] = M''ₓ(0)
M''ₓ(t) = n(n−1) (q + p·eᵗ)ⁿ⁻² (p·eᵗ)² + n (q + p·eᵗ)ⁿ⁻¹ p·eᵗ
M''ₓ(0) = n(n−1) p² + np

Step 3 — Variance
Var(X) = M''ₓ(0) − [M'ₓ(0)]²
      = n(n−1)p² + np  −  n²p²
      = np − np² = np(1−p) = npq         ✓

Answer: E[X] = np,  Var(X) = npq.

🔑 "Differentiate, plug 0, get moment" — MGF rule in 6 words.

✅ Model Exam Answer

Q6 Joint Distribution — discrete & continuous

⚡ Core Intuition

Joint distribution answers "what's the probability of both X and Y at once" — needed when outcomes pair up.

📖 Concept Explained
  • Joint pmf (discrete): p(x,y) = P(X=x, Y=y), Σ Σ p(x,y) = 1.
  • Joint pdf (continuous): f(x,y) ≥ 0, ∬ f(x,y) dx dy = 1.
  • Joint CDF: F(x,y) = P(X≤x, Y≤y).
  • F is monotone non-decreasing in each variable.
  • F(−∞,y)=F(x,−∞)=0; F(∞,∞)=1.

  • f(x,y) = ∂² F(x,y) / ∂x ∂y (continuous).

Q8 Conditional Distribution — joint ÷ marginal

⚡ Core Intuition

Once you know X=x, the universe shrinks to that row — rescale by the row sum to get the conditional probabilities of Y.

📖 Concept Explained
  • Discrete: P(Y=y | X=x) = p(x,y) / p_X(x), provided p_X(x) > 0.
  • Continuous: f_{Y|X}(y|x) = f(x,y) / f_X(x).
  • Conditional pmf/pdf sums/integrates to 1 (over the conditioned variable).
  • E[Y|X=x] = Σ y · P(Y=y|X=x).
  • Law of total expectation: E[Y] = E[E[Y|X]].
🔗 Connects to marginal Bayes independence
📐 Exam Answer Structure
  • Define conditional pmf/pdf.
  • Show: rows of joint table, divided by row sum.
  • Verify Σ P(Y=y|X=x) = 1.
  • Compute E[Y|X=x] as bonus.
💡 Example

Given X=1: P(Y=0|X=1) = (2/12)/(5/12) = 2/5; rescaling preserves total = 1.

Continuous: f(x,y) = (x+y)/8,  0 ≤ x,y ≤ 2 ;  f_X(x) = (x+1)/4
Conditional f_{Y|X}(y|x) = f(x,y) / f_X(x)
                        = [(x+y)/8] / [(x+1)/4]
                        = (x+y) / [2(x+1)],   0 ≤ y ≤ 2

Check ∫₀² f_{Y|X}(y|1) dy
     = ∫₀² (1+y)/(2·2) dy
     = (1/4)[y + y²/2]₀² = (1/4)(2+2) = 1     ✓

E[Y | X=1] = ∫₀² y · (1+y)/4 dy
           = (1/4)[y²/2 + y³/3]₀²
           = (1/4)(2 + 8/3) = 7/6 ≈ 1.17

🔑 "Joint over Marginal" — the conditional recipe.

  2. Diagram: row of joint ÷ row sum (above).
  3. Verify conditional sums to 1.
  4. Compute E[Y|X=x] (numerical above).

Q9 Independence of Random Variables

⚡ Core Intuition

X and Y are independent iff knowing X tells you nothing about Y — equivalently, the joint factorises into the product of marginals.

📖 Concept Explained
  • Definition (discrete): X⊥Y iff p(x,y) = p_X(x)·p_Y(y) for all (x,y).
  • Continuous: X⊥Y iff f(x,y) = f_X(x)·f_Y(y) for all (x,y).
  • CDF form: X⊥Y iff F(x,y) = F_X(x)·F_Y(y).
  • If X⊥Y: E[XY] = E[X]·E[Y], Cov(X,Y) = 0.
  • If X⊥Y: M_{X+Y}(t) = Mₓ(t)·M_Y(t).
  • Converse of "Cov = 0" is FALSE — uncorrelated ≠ independent.
🔗 Connects to joint covariance MGF
📐 Exam Answer Structure
  • Define independence (joint = product of marginals).
  • Pick any one cell — check joint vs marginal product.
  • If any cell fails, declare not independent.
  • State consequences for E[XY] and Var.
💡 Example

Two fair dice: p(x,y)=1/36 = (1/6)·(1/6) = p_X(x)·p_Y(y) for all cells ⇒ independent.

Continuous test: f(x,y) = 4xy,  0 ≤ x,y ≤ 1
Marginals:
  f_X(x) = ∫₀¹ 4xy dy = 2x,    0 ≤ x ≤ 1
  f_Y(y) = ∫₀¹ 4xy dx = 2y,    0 ≤ y ≤ 1
Product:
  f_X(x)·f_Y(y) = (2x)(2y) = 4xy = f(x,y)  ✓ for all (x,y)
Conclusion: X ⊥ Y are INDEPENDENT.

Contrast: f(x,y) = (x+y)/8 over [0,2]² :
  f_X(x) = (x+1)/4,  f_Y(y) = (y+1)/4
  f_X·f_Y = (x+1)(y+1)/16  ≠  (x+y)/8 in general
Conclusion: NOT independent.

🔑 "Joint = Product" — independence in 3 words.

✅ Model Exam Answer
  1. Definition: X⊥Y iff joint = product of marginals.
  2. Compute both marginals from joint.
  3. Pick any cell — test equality (above).
  4. If one fails, declare dependent.
  5. State consequences (E[XY], Cov, MGF factorisation).
  6. Warn: Cov=0 ⇏ independence.

STANDARD MGFs (memorise)
  Bernoulli(p):    M(t) = q + p eᵗ
  Binomial(n,p):   M(t) = (q + p eᵗ)ⁿ          E=np,  Var=npq
  Poisson(λ):      M(t) = exp(λ(eᵗ − 1))       E=λ,   Var=λ
  Geometric(p):    M(t) = p eᵗ /(1 − q eᵗ)     E=1/p, Var=q/p²
  Uniform(a,b):    M(t) = (e^(bt) − e^(at))/[t(b−a)]
  Exponential(λ):  M(t) = λ/(λ − t),  t<λ      E=1/λ, Var=1/λ²
  Normal(μ,σ²):    M(t) = exp(μt + σ²t²/2)

JOINT / MARGINAL / CONDITIONAL
  Joint CDF        F(x,y) = P(X≤x, Y≤y)
  Joint pdf        f(x,y) = ∂²F/∂x∂y
  Σ Σ p(x,y) = 1   ;   ∬ f(x,y) dx dy = 1
  Marginal X:  p_X(x) = Σ_y p(x,y)   /   f_X(x) = ∫ f(x,y) dy
  Marginal Y:  p_Y(y) = Σ_x p(x,y)   /   f_Y(y) = ∫ f(x,y) dx
  Conditional: f_{Y|X}(y|x) = f(x,y) / f_X(x)
  Rect prob:   P(a<X≤b, c<Y≤d) = F(b,d) − F(a,d) − F(b,c) + F(a,c)

INDEPENDENCE
  Discrete  : p(x,y) = p_X(x)·p_Y(y)    ∀(x,y)
  Continuous: f(x,y) = f_X(x)·f_Y(y)    ∀(x,y)
  CDF       : F(x,y) = F_X(x)·F_Y(y)
  Then      : E[XY]=E[X]E[Y],  Cov=0,  Var(X+Y)=Var(X)+Var(Y),
              M_{X+Y}(t) = Mₓ(t)M_Y(t)

🚨 Common mistakes (red flags)

  • Writing E[XY] = E[X]·E[Y] without first proving independence — only holds when X⊥Y.
  • Treating Cov(X,Y)=0 as independence — they are NOT equivalent (only for jointly normal).
  • Forgetting μ₂ = μ'₂ − μ'₁², not just μ'₂. Subtract the squared mean.
  • Stating kurtosis as μ₄/σ⁴ for "excess" — excess kurtosis is μ₄/σ⁴ − 3.
  • Differentiating MGF and forgetting to evaluate at t = 0 — that's the whole point.
  • Confusing PGF (z-domain, discrete only) with MGF (eᵗ, all RVs).
  • Marginal = row OR column total — students reverse them under exam pressure.
  • Conditional P(Y|X) ≠ P(X|Y) — different denominator, different distribution.
  • Testing independence with only one cell when it equals — must hold for ALL (x,y).
  • Forgetting joint pdf must satisfy f ≥ 0 AND ∬ f = 1 — both checks needed.

Q4 Compute the standard deviation and variance for the following grouped frequency distribution.

⚡ Core Intuition

  • Square removes signs; mean of squares = variance σ².

Q7 Define kurtosis. Differentiate leptokurtic, mesokurtic and platykurtic curves with diagram.

⚡ Core Intuition

Skewness asks "lopsided?" Kurtosis asks "pointy or flat?" High kurtosis = sharp peak + fat tails. Low kurtosis = squashed, even spread.

  • Mesokurtic β₂ = 3 (normal-like).
  • Platykurtic β₂ < 3 (flatter, thin tails).

🚨 Common mistakes (prof penalties)

  • Applying ungrouped formula (Σx/n) to grouped data — must use Σfx/Σf with class midpoints.
  • Using upper class limit as L in median/mode formulas. L = lower limit of the class.
  • Forgetting Bessel correction (n−1) when asked for sample SD.
  • Confusing skewness with kurtosis. Skew = lopsided; Kurt = pointy/flat.
  • Drawing histogram with gaps. Histograms have NO gaps — bar charts do.
  • Quoting Pearson Sk without σ in denominator — formula is (Mean−Mode)/σ.
  • Sign error in covariance — keep deviations (x−x̄), (y−ȳ) signed; product carries the answer.
  • Saying "Cov > 1 means strong relation". Wrong — Cov is unbounded. Use r for strength.
  • Treating mode as always unique. Distributions can be bi-modal / multi-modal.
  • Calling nominal data "ordered". Nominal = labels only; ordering them is illegal.

💡 Karpathy intuition (memorise verbatim)

"Mean = balance point. Median = middle line. Mode = peak.

  • For skewed data, median is most robust — it ignores the long tail.
  • Mean follows the tail; mode stays at the peak; median sits between.
  • Right-skew (positive): Mean > Median > Mode.
  • Left-skew (negative): Mean < Median < Mode."

Q2 Compute Karl Pearson's coefficient of correlation r for the data: x = (1,2,3,4,5), y = (2,5,3,8,7).

⚡ Core Intuition

Pearson r = covariance normalised by the product of standard deviations. Subtract the means so deviations carry the signal; multiply x and y deviations, sum them up; divide by the geometric mean of the squared-deviation sums.

📖 Concept Explained
  • r measures linear association only.
  • Unit-free — dividing by σₓσᵧ cancels all units.
  • r ∈ [−1, +1]; +1 perfect line up, −1 perfect line down, 0 no linearity.
  • Symmetric: r(x,y) = r(y,x). Unaffected by linear scaling/origin shift.
  • Sensitive to outliers and assumes linearity.

  • Compute Σ for each column.
  • Substitute → simplify √ → final r with interpretation.

Q3 Compute Spearman's rank correlation coefficient rs for the marks of 5 students in Maths (85, 60, 73, 40, 90) and Stats (78, 55, 80, 50, 92).

⚡ Core Intuition

Replace raw values with their ranks, then ask: do the rank orders agree? If yes → rs close to +1. If reversed → −1. Robust to outliers because ranks compress them.

📖 Concept Explained
  • Use when data are ordinal or non-linear monotonic.
  • Rank each variable separately (1 = highest, 2 = next…).
  • d = difference of paired ranks; square and sum.
  • rs ∈ [−1, +1] just like Pearson r.
  • For tied ranks, assign average rank and add a correction term.

Q4 Define regression analysis. Write the two regression lines, explain the regression coefficients byx, bxy and the angle between them.

⚡ Core Intuition

Correlation says how strong. Regression says predict: given x, what's the best guess of y? Two lines exist because predicting y-from-x ≠ predicting x-from-y unless r = ±1.

📖 Concept Explained
  • Regression line of Y on X minimises Σ(y − ŷ)² (vertical errors).
  • Regression line of X on Y minimises Σ(x − x̂)² (horizontal errors).
  • byx = slope of Y-on-X = r·σᵧ/σₓ.
  • bxy = slope of X-on-Y = r·σₓ/σᵧ.
  • Both lines pass through (x̄, ȳ).
  • If r = ±1 the two lines coincide; if r = 0 they are perpendicular.
🔗 Connects to Least squares Pearson r r² = byx·bxy Prediction
📐 Exam Answer Structure
  • State both regression equations.
  • Define byx and bxy with formulas.
  • Diagram showing the two lines and their intersection at (x̄, ȳ).
  • Properties: range, sign, angle, r relation.
Two regression lines

   Y on X :   y − ȳ = b_yx · (x − x̄)
              b_yx = Σ(x−x̄)(y−ȳ) / Σ(x−x̄)²
                   = r · σ_y / σ_x

   X on Y :   x − x̄ = b_xy · (y − ȳ)
              b_xy = Σ(x−x̄)(y−ȳ) / Σ(y−ȳ)²
                   = r · σ_x / σ_y

Properties
   sign(b_yx) = sign(b_xy) = sign(r)
   |b_yx · b_xy| = r²   so  −1 ≤ r ≤ +1
   Both lines pass through (x̄, ȳ).
   Angle θ between the two lines:
       tan θ = ((1 − r²) / |r|) · (σ_x σ_y) / (σ_x² + σ_y²)
💡 Example

Predicting marks (Y) from hours studied (X): use Y-on-X. Predicting hours from marks: use X-on-Y. The two equations are not algebraic inverses of each other.

Mnemonic — "Predicted-variable owns the line": "Y on X" means Y is the one being predicted; minimise errors in Y.
✅ Model Exam Answer

Q6 State the principle of least squares. Fit a straight line y = a + bx to x = (1,2,3,4,5), y = (2,5,3,8,7) by least squares.

⚡ Core Intuition

Of all possible lines, the least-squares line is the one whose total squared vertical distance to the data is smallest. Squaring kills the sign and punishes big errors more — so the line cannot drift too far from any point.

📖 Concept Explained
  • Residual ei = yi − (a + bxi).
  • Choose a, b that minimise S = Σei².
  • Setting ∂S/∂a = 0 and ∂S/∂b = 0 gives normal equations.
  • Solve the two simultaneous equations → unique a, b.
  • Best line always passes through (x̄, ȳ).

🚨 Common mistakes (prof penalties)

  • Forgetting to subtract the correction (mean) before squaring — using raw Σx² instead of Σ(x−x̄)² gives a totally wrong r.
  • Writing r = byx·bxy. Wrong — it is r2 = byx·bxy. Always take square root and put correct sign.
  • Quoting r outside [−1, +1]. Impossible — recheck arithmetic if it happens.
  • Treating both regression lines as algebraic inverses of each other. They are not unless r = ±1.
  • In Spearman, ranking from low → high instead of high → low (or vice versa) for only one variable. Be consistent across both.
  • In Spearman with ties, forgetting the correction term T = (t³ − t)/12.
  • Writing normal equations with Σx in the wrong slot — for line: row 1 has Σx, row 2 has Σx². Order matters.
  • For parabola, summing only up to Σx² — you need Σx³ and Σx⁴ as well.
  • For power curve, taking log of only y (it's exponential then). Take log of both sides for power, only y for exponential.
  • Forgetting to convert B back to b in exponential fit — b = B / log e = B · 2.3026, not B itself.
  • Concluding "r = 0 ⇒ X, Y independent." Wrong — r = 0 only rules out linear dependence.
  • Confusing covariance with correlation — Cov is unit-dependent and unbounded; r is unit-free and bounded.

💡 Karpathy intuition (memorise verbatim)

"Correlation = how strongly do x and y move together?

  • r = +1 perfect positive line · r = −1 perfect negative line · r = 0 no linear relation.
  • Regression line is the line that minimises squared distances — out of all infinitely many lines, this one is the unique best.
  • If you can predict y from x, you can almost predict x from y — but not exactly. That's why there are two regression lines.
  • Log is the magic trick: it bends a curve until it becomes straight, so the simple line-fit machinery still works."

Q. If byx = 1.6 and bxy = 0.4, find r.
A. r² = 1.6·0.4 = 0.64 ⇒ r = +0.8 (positive since both slopes are positive).
Q. Two regression lines coincide. What is r?
A. r = ±1 (perfect linear relation — the same line minimises both error directions).
Q. r = 0 but the scatter is U-shaped. What does it mean?
A. No linear association; a quadratic relation may still exist. Pearson misses non-linear patterns.
Q. How many normal equations to fit y = a + bx + cx² + dx³?
A. Four — one per parameter (a, b, c, d). General rule: one equation per free parameter.
Q. Convert y = 5·e^(0.7x) into a linear form.
A. log y = log 5 + 0.7·log e · x = 0.699 + 0.304 x ⇒ a straight line in (x, log y).
Q. For Spearman with no ties, what Σd² makes rs = 0 when n = 5?
A. 0 = 1 − 6Σd²/(5·24) ⇒ Σd² = 20.
Q. Why do both regression lines pass through (x̄, ȳ)?
A. The normal equations force the mean of fitted ŷ to equal ȳ — so the centroid is on every least-squares line.

 STANDARD  DISCRETE  DISTRIBUTIONS   (Unit 1)
============================================================
 BINOMIAL  X ~ B(n,p)        q = 1 - p

   Mean     = n p
   Variance = n p q

 POISSON   X ~ Poi(λ)            (λ > 0)

 GEOMETRIC X ~ Geo(p)        (trial-until-first-success)

 STANDARD  CONTINUOUS  DISTRIBUTIONS   (Unit 1)
============================================================
 UNIFORM     X ~ U(a,b)
   f(x)  = 1/(b-a)   for a ≤ x ≤ b
   Mean  = (a+b)/2     Var = (b-a)^2 / 12

 EXPONENTIAL X ~ Exp(λ)         (λ > 0)
   f(x)  = λ e^(-λ x)    x ≥ 0
   F(x)  = 1 - e^(-λ x)
   Mean  = 1/λ
   Var   = 1/λ^2
   MGF   = λ / (λ - t)        (t < λ)

 NORMAL      X ~ N(μ, σ^2)
   f(x)  = (1 / (σ √(2π))) · exp[ -(x-μ)^2 / (2σ^2) ]
   Mean  = μ       Var = σ^2
   MGF   = exp[ μ t + ½ σ^2 t^2 ]
   Standardise:  Z = (X - μ) / σ   ~  N(0, 1)
   Empirical rule:  68 % | 95 % | 99.7 %    in μ±1,2,3 σ

 MOMENT  GENERATING  FUNCTION   (Unit 2)
============================================================
 Definition         MX(t) = E[ e^(tX) ]

 Property 1   MX(0)        = 1

 JOINT  /  MARGINAL  /  CONDITIONAL   (Unit 2)
============================================================
 Joint pmf      p(x,y) ≥ 0     ΣΣ p(x,y) = 1
 Joint pdf      f(x,y) ≥ 0     ∫∫ f(x,y) dx dy = 1

 Marginal X     fX(x)  = Σy p(x,y)    or   ∫ f(x,y) dy
 Marginal Y     fY(y)  = Σx p(x,y)    or   ∫ f(x,y) dx

 Conditional    f(y | x) = f(x,y) / fX(x)     (fX(x) > 0)

 Independence   f(x,y) = fX(x) · fY(y)   for all x,y

 E[XY]          = ΣΣ x y p(x,y)  or  ∫∫ xy f(x,y) dx dy
 X ⊥⊥ Y          =>  E[XY] = E[X] E[Y]  ,  Cov = 0

============================================================
 CENTRAL  TENDENCY  —  GROUPED  DATA   (Unit 3)
============================================================
 Mean         x̄   = Σ f x / Σ f       (x = class mid-point)

 Step-deviation
              x̄   = A + ( Σ f d / Σ f ) · h
              d   = (x - A) / h

 Median       M   = L + [ (N/2 - cf) / f ] · h
              L   = lower limit of median class
              cf  = cumulative freq before median class
              f   = freq of median class
              h   = class width
              N   = Σ f

 Mode         Mo  = L + [ (f1 - f0) / (2 f1 - f0 - f2) ] · h

 Empirical    Mode ≈ 3 · Median - 2 · Mean

============================================================
 DISPERSION  &  SD  —  GROUPED  DATA   (Unit 3)
============================================================
 Range         = Xmax - Xmin
 Mean Deviation MD = Σ f |x - x̄| / Σ f

 Variance      σ² = Σ f (x - x̄)² / Σ f
                  = ( Σ f x² / Σ f ) - x̄²          (shortcut)

 SD            σ  = √ [ Σ f (x - x̄)² / Σ f ]

 Sample SD     s  = √ [ Σ f (x - x̄)² / (n - 1) ]   (Bessel)

 Coeff. Var.   CV = (σ / x̄) · 100

 Skewness (Karl-Pearson)
              Sk = (Mean - Mode) / SD
              Sk = 3 (Mean - Median) / SD     (when mode unstable)

 Bowley       SkB = (Q3 + Q1 - 2 Q2) / (Q3 - Q1)

============================================================
 Cov(X,Y)      = E[XY] - E[X] E[Y]
                = (1/n) Σ (x - x̄)(y - ȳ)
                = (1/n) Σ xy - x̄ ȳ

 Karl-Pearson r (product-moment)
              r = Cov(X,Y) / (σx σy)

                  n Σxy  -  Σx Σy
              r = ----------------------------------------------
                  √[ n Σx² - (Σx)² ]  ·  √[ n Σy² - (Σy)² ]

  Range  -1 ≤ r ≤ +1
   r =  +1  perfect positive
   r =  -1  perfect negative
   r =   0  no LINEAR relation (could still be related)

 Spearman rank (ties absent)
              rs = 1  -   6 Σ d²  /  ( n ( n² - 1 ) )
              d = rankX - rankY

 Spearman (ties present)  add  Σ ( m³ - m ) / 12   inside the bracket.

 REGRESSION  LINES   (Unit 4)
============================================================
 Line of Y on X    y - ȳ = byx (x - x̄)
 Line of X on Y    x - x̄ = bxy (y - ȳ)

 Slopes (regression coefficients)
              byx =  r · (σy / σx)  =  Cov(X,Y) / σx²
              bxy =  r · (σx / σy)  =  Cov(X,Y) / σy²

 Identity     r² = byx · bxy          (sign of r = sign of slopes)

 Angle between lines  tanθ = | (1 - r²) / r |  ·  (σx σy) / (σx² + σy²)
   r = ±1   lines coincide
   r =  0   lines perpendicular

============================================================
 CURVE  FITTING  —  LEAST  SQUARES   (Unit 4)
============================================================
 STRAIGHT  LINE       y = a + b x
   Normal equations
        Σ y      = n a + b Σ x
        Σ x y    = a Σ x + b Σ x²

Top 20 — One-Mark Drill

Tap to reveal. Aim < 10 sec per Q.

Q1 · State Bayes' theorem.
P(A|B) = P(B|A)·P(A) / P(B), with P(B) = Σ P(B|Aᵢ)P(Aᵢ).
Q2 · Mean and variance of Binomial(n,p)?
Mean = np · Variance = npq (q = 1−p).
Q3 · Mean and variance of Poisson(λ)?
Mean = Variance = λ. Identity that flags Poisson on sight.
Q4 · PDF of Exponential(λ)?
f(x) = λ e−λx, x ≥ 0. Mean = 1/λ, Var = 1/λ2.
Q5 · Memoryless property statement?
P(X > s+t | X > s) = P(X > t). Only Exponential (continuous) & Geometric (discrete).
Q6 · Standardisation formula for Normal?
Z = (X − μ) / σ — converts N(μ,σ2) to N(0,1).
Q7 · What does MGF stand for — definition?
Moment Generating Function MX(t) = E[etX]. Derivatives at 0 give moments.
Q8 · MGF of Poisson(λ)?
exp[λ(et−1)].
Q9 · MGF of Binomial(n,p)?
(q + pet)n.
Q10 · How to get marginal from joint?
Sum (discrete) or integrate (continuous) out the other variable.
Q11 · Condition for two RVs to be independent?
f(x,y) = fX(x)·fY(y) for ALL (x,y). Also E[XY]=E[X]E[Y] when indep.
Q12 · Variance formula in shortcut form?
Var(X) = E[X2] − (E[X])2.
Q13 · Median of grouped data formula?
M = L + [(N/2 − cf) / f] · h.
Q14 · Empirical relation between Mean, Median, Mode?
Mode ≈ 3·Median − 2·Mean (Karl-Pearson empirical rule).
Q15 · SD vs Variance — relation?
SD σ = √Variance. SD has same units as data, variance has squared units.
Q16 · Karl-Pearson coefficient of skewness?
Sk = (Mean − Mode)/SD — or 3(Mean − Median)/SD.
Q17 · Cov(X,Y) when X, Y are independent?
0. (Reverse is NOT true — zero cov does not imply independence.)
Q18 · Range of correlation coefficient r?
−1 ≤ r ≤ +1. r = 0 means no LINEAR relation, not no relation.
Q19 · Spearman rank formula?
rs = 1 − 6Σd2/[n(n2−1)], d = rank difference.
Q20 · Relationship between r, byx, bxy?
r2 = byx·bxy. Sign of r = common sign of the two slopes.

Red Flags — Common Penalties

Avoid these — each one costs 1–2 marks even when answer is "correct".

  • Using n vs (n−1) wrong — population SD divides by n, sample SD divides by (n−1). Read the question.
  • Mixing up byx and bxy — byx = r·σy/σx for the Y-on-X line; swap the σ's for the other line.
  • Writing r² = (byx+bxy)/2 — wrong. r² = byx · bxy (product).
  • r = 0 means "independent" — NO. Means no LINEAR relation. Two RVs can have r=0 and still be dependent.
  • Forgetting q = 1−p in Binomial — variance = npq, not np² or np.
  • Confusing Poisson mean/var — both equal λ. If they're different, it isn't Poisson.
  • Normal P(X < x) without Z-transform — always standardise first; quote Φ(z) from the table.
  • Skipping cumulative-frequency check for median — the median class is the FIRST one whose cf ≥ N/2.
  • Mode formula with wrong f0, f1, f2 — f1 = modal class frequency, f0 = preceding, f2 = succeeding.
  • Karl-Pearson formula sign errors — (nΣx² − (Σx)²) inside the root, NOT (Σx² − n(Σx)²).
  • Spearman with ties uncorrected — add Σ(m³−m)/12 inside the bracket for every tied group.
  • Normal equations forgotten in curve fitting — line has 2 equations, parabola has 3. Drop one → system unsolvable.
  • No diagram in a 15-mark answer — scatter / bell / skewness shapes cost zero time and add 2–3 marks.
  • Treating MGF as a number — MGF is a function of t. Mean = M'(0), Variance = M''(0) − (M'(0))².
  • Forgetting domain in Exponential — pdf is λe−λx for x ≥ 0, zero elsewhere. Always state the support.
Final pitfall — Read the question twice. Many lose marks by computing r when "regression lines" were asked, or vice versa.

Topic-Wise Consolidation

Same concepts reappear in slightly different wrappers — group them, master once.

① Joint PDF — Marginal + Conditional (the king topic)

B.Tech EL Q4 [15], B.Tech EL Q1(c) [1.5], B.Sc Q1(h) [1.5], B.Sc Q4(b) [8], B.Sc Q7(a) [4]

  • Always: ∫∫ f(x,y) dx dy = 1 over support.
  • Marginal: f_X(x) = ∫ f(x,y) dy over y-range; f_Y(y) similarly.
  • Conditional: f(y|x) = f(x,y) / f_X(x).
  • Common forms asked: f=2 (rectangular), f=2−x−y, f=(8/9)xy, f=(6−x−y)/8.

② Karl Pearson Correlation Coefficient

B.Tech CE Q5(a) [8], B.Tech EL Q7 [15], B.Sc Q5(a) [8], B.Sc Q7(c) [6]

  • Formula: r = Σ(x−x̄)(y−ȳ) / √[Σ(x−x̄)² · Σ(y−ȳ)²].
  • Short-cut method using deviations from assumed mean — much faster in exam.
  • Always asked with 8–12 data pairs — table format.
  • Property: r unchanged by origin/scale shift — proved in B.Tech CE Q5(b).

③ Regression — Two Lines (Y on X & X on Y)

B.Tech CE Q4(a) [8], B.Tech EL Q1(f) [1.5], B.Tech EL Q6(b), B.Sc Q1(j) [1.5], B.Sc Q5(b) [7]

  • Y on X: y − ȳ = b_yx (x − x̄), where b_yx = r·(σ_y/σ_x).
  • X on Y: x − x̄ = b_xy (y − ȳ), where b_xy = r·(σ_x/σ_y).
  • Relation: r² = b_yx × b_xy ⇒ r = √(b_yx · b_xy).
  • Asked in 3 papers — guaranteed appearance.

④ Normal Distribution + Properties

B.Tech EL Q3(b) [10], B.Tech EL Q1(e) [1.5], B.Sc Q1(e) [1.5], B.Sc Q3(b) [7]

  • Mean=μ, variance=σ²; mode=median=mean=μ.
  • Standard Normal: Z=(X−μ)/σ, mean 0, var 1.
  • B.Sc Q3(b) recurring: bank account problem — convert to Z, use table.
  • 3 properties to recall: symmetric, bell-shaped, total area=1, ±σ ≈ 68%.

⑤ Binomial Distribution — params, MGF, SD

B.Tech EL Q1(d) [1.5], B.Tech EL Q5(a) [5], B.Sc Q1(d) [1.5], B.Sc Q3(a) [8]

  • Mean = np, Variance = npq, SD = √(npq).
  • MGF M(t) = (q + pe^t)ⁿ.
  • Mode = ⌊(n+1)p⌋ usually.
  • Reverse problems: given mean+var, find n, p (Q1(d) both papers).

⑥ Moment Generating Function (MGF)

B.Tech EL Q5(a) [5], B.Sc Q1(g) [1.5], B.Sc Q2(c) [7]

  • Definition: M_X(t) = E[e^(tX)] — discrete sum or continuous integral.
  • k-th moment: M^(k)(0) = E[X^k].
  • MGF of Binomial, Poisson, Normal, Exponential — memorize.
  • B.Sc Q2(c): uniform on (−1,2) and triangular — direct integration.

⑦ Continuous PDF — find k / probabilities

B.Tech CE Q7(b) [7], B.Sc Q1(b) [1.5], B.Sc Q2(b) [4]

  • Normalize: ∫ f(x) dx = 1 over support → solve for K.
  • Mean = ∫ x·f(x) dx, Variance = E(X²) − μ².
  • For piecewise f(x), split integral by ranges.
  • P(X≤a)=P(X>a) ⇒ both = 0.5 (median).

⑧ Skewness & Kurtosis

B.Tech CE Q1(b) + Q1(f) [3], B.Tech EL Q1(j) [1.5]

  • Skewness = asymmetry; γ₁ = μ₃/σ³ (positive/negative/zero).
  • Kurtosis = peakedness; β₂ = μ₄/σ⁴.
  • β₂ > 3 → leptokurtic (sharp peak); β₂ < 3 → platykurtic (flat); =3 → mesokurtic.
  • Bowley's, Pearson's, Kelly's measures of skewness.

⑨ Bayes / Conditional Probability

B.Tech CE Q7(a) [8], B.Sc Q6(a) [6]

  • P(A|B) = P(B|A)·P(A) / P(B).
  • B.Tech: insurance/driver type given accident (3 categories).
  • B.Sc: missing card heart given 2 hearts drawn.
  • Always: set up P(A_i) priors → compute Σ P(B|A_i)P(A_i) → divide.

⑩ Curve Fitting — Least Squares (line / parabola / exponential)

B.Tech CE Q4(b) [7], B.Tech EL Q1(g) [1.5], B.Tech EL Q6(a) [10], B.Sc Q5(b) [7]

  • Line y=a+bx: solve Σy=na+bΣx, Σxy=aΣx+bΣx².
  • Parabola y=a+bx+cx²: 3 normal equations.
  • Exponential y=ab^x: take log → log y = log a + x·log b (linear in x).
  • Always exam pattern: table of (x,y) points → compute sums → solve.
